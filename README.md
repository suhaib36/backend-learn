Backend Learn

A backend learning repository built using Node.js, Express.js, and MongoDB to practice backend development concepts including database connection, APIs, routing, and server-side logic.

Tech Stack
Node.js
Express.js
MongoDB Atlas
Mongoose
Features
MongoDB database connection
Express server setup
REST API practice
Backend routing
Environment configuration
Modular project structure
Installation

Clone the repository:

git clone https://github.com/suhaib36/backend-learn.git

Move into project folder:

cd backend-learn

Install dependencies:

npm install

Run server:

node server.js
Project Structure
backend-learn/
│── node_modules/
│── server.js
│── package.json
│── package-lock.json
│── .gitignore

Understand backend architecture
